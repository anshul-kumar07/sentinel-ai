# Sentinel AI

Sentinel AI is a context-aware scam risk assistant that helps users
identify hidden fraud signals in messages, emails, and links.

## Features
- Analyzes pasted text for scam risk
- Detects lookalike domains
- Explains why a message is risky
- Supports job, bank, and internship scams

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML / JavaScript
- AI Logic: Rule + prompt-based reasoning
- Cloud Architecture (Planned): Google Cloud (Vertex AI, Cloud Functions)

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run backend:
   python app.py
4. Open frontend/index.html in browser

## Note
This is a hackathon prototype. Backend execution is local.
